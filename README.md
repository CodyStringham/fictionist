Ruby on Rails
-------------

This application requires:

- Ruby 2.2.0
- Rails 4.2.0

Learn more about [Installing Rails](http://railsapps.github.io/installing-rails.html).

Getting Started
---------------

- Clone this repo
- Run `bin/setup` and follow instructions.
- Run `rails s`
